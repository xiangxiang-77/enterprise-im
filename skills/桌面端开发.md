# 桌面端开发 Skill

## 技术栈
- **框架**: Qt 5.9.3 (msvc2017_64, WebEngine)
- **编译**: Visual Studio 2017 BuildTools + nmake
- **语言**: C++17 (受限于 MSVC 14.16)
- **媒体**: PJSIP/PJSUA (外部进程, SIP+RTP), Qt Multimedia
- **存储**: QSettings + QSqlDatabase (SQLite 本地缓存)
- **网络**: QTcpSocket + QNetworkAccessManager

## 项目结构
```

├── src/
```

## 架构特点

- `MainWindow.cpp` 约 4300 行，包含所有 UI 构建和逻辑
- 构造函数中一次性构建所有 Widget（登录面板、左面板、聊天区、通话面板）

## 开发约束

### C++ 版本限制 (MSVC 2017)
- 禁止 `auto` 结构化绑定: `auto [a, b] = tuple;`
- 禁止 `if constexpr`、折叠表达式
- 禁止 `std::optional`、`std::variant`
- lambda 只能是基本形式

### Qt 5.9 信号/槽
```cpp
// 正确: 函数指针语法，参数完全匹配
connect(btn, &QPushButton::clicked, this, &MainWindow::onClick);

// 错误: 带默认参数的槽无法直接连接
// connect(edit, &QLineEdit::returnPressed, this, &MainWindow::search);  // 编译失败
// 解决: 使用 lambda 包装
connect(edit, &QLineEdit::returnPressed, this, [this]() { search("all"); });
```

### 包含文件
- Qt Widget 类需显式 `#include <QXxx>`（禁止依赖间接包含）
- 新增 Widget 使用前必须添加对应 `#include`
