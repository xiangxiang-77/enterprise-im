## Qt 特有设计约束
- Qt 5.9 C++ 标准: 禁用 `auto` 模板推导、结构化绑定
- 信号/槽: 用函数指针语法 `&Class::method`，参数类型必须完全匹配
- UI 布局: 手动 C++ 代码布局，禁止 `.ui` 文件
- 线程: 网络操作在主线程，耗时操作用 QtConcurrent

## Flutter 特有设计约束
- 状态管理: StatefulWidget，不使用 Provider/Bloc
- 路由: Navigator.push/pop，不使用 go_router
- 包管理: 依赖声明在 `dependencies` 非 `dependency_overrides`
