## 技术栈
- **框架**: React 18 + TypeScript
- **构建**: Vite 7
- **UI**: Tailwind CSS + shadcn/ui (Radix UI)
- **图表**: recharts (AreaChart, BarChart)
- **路由**: react-router-dom v6
- **状态**: Zustand (轻量级 Store)
- **HTTP**: fetch (通过 api.ts 封装)

## 开发规范

### API 调用模式
```typescript
// api.ts — 函数式封装
export async function getAdminDashboardApi(token: string): Promise<AdminDashboard> {
  return get<AdminDashboard>("/api/admin/dashboard", token)
}

// 页面中使用
const [data, setData] = useState<AdminDashboard | null>(null)
useEffect(() => {
  if (session?.token) getAdminDashboardApi(session.token).then(setData)
}, [session])
```

### 组件模式
```tsx
function MyPage() {
  const [loading, setLoading] = useState(true)
  const [data, setData] = useState<DataType[]>([])
  const { session } = useAuthStore()

  useEffect(() => { load() }, [session])

  async function load() {
    if (!session?.token) return
    setLoading(true)
    try {
      const result = await fetchDataApi(session.token)
      setData(result)
    } catch (e) {
      toast.error("加载失败")
    } finally {
      setLoading(false)
    }
  }

  if (loading) return <LoadingSpinner />
  return (/* ... */)
}
```

// 按角色分离: /admin/* (管理端), /chat/* (用户端)

### 样式约定
- Tailwind CSS 原子类优先
- shadcn/ui 组件通过 `npx shadcn-ui@latest add` 安装
- 主题色: `primary` (蓝色系), `destructive` (红色系)
- 响应式: `sm:` `md:` `lg:` 断点

### 新页面 Checklist
