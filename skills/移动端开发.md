# 移动端开发 Skill

## 技术栈
- **框架**: Flutter 3.x (Dart, SDK ≥3.0.0)
- **平台**: Android (PJSIP2 JNI 集成)
- **状态管理**: StatefulWidget (无 Provider/Bloc/Riverpod)
- **存储**: SharedPreferences + sqflite
- **路由**: Navigator.push/pop (无 go_router)
- **包依赖**: `pubspec.yaml` dependencies 段

## 开发规范

### 状态管理
```dart
class MyScreen extends StatefulWidget { ... }

class _MyScreenState extends State<MyScreen> {
  List<Item> _items = [];
  bool _loading = true;

  @override
  void initState() {
    super.initState();
    _load();
  }

  Future<void> _load() async {
    setState(() => _loading = true);
    try {
      final data = await ApiService.getItems();
      setState(() { _items = data; _loading = false; });
    } catch (e) {
      setState(() => _loading = false);
    }
  }

  @override
  Widget build(BuildContext context) { ... }
}
```

### API 调用
```dart
// api_service.dart
static Future<List<User>> getContacts() async {
  final prefs = await SharedPreferences.getInstance();
  final token = prefs.getString('token') ?? '';
  final response = await http.get(
    Uri.parse('$baseUrl/api/contacts'),
    headers: {'Authorization': 'Bearer $token'},
  );
  final body = jsonDecode(response.body);
  return (body['data'] as List).map((j) => User.fromJson(j)).toList();
}
```

### 消息类型渲染
- 文本: RichText + URL 正则 → url_launcher
- 图片: Image.network + 网格布局 (1-9张)
- 文件: 文件名 + 大小 + 下载按钮
- 语音: 播放按钮 + audioplayers
- 视频: 缩略图 + video_player
- 名片: 头像 + 名称卡片
- 系统消息: 图标 + 特殊样式 (创建/加人/移除/公告)
- @提及: 黄色/蓝色高亮背景
- 阅后即焚: 倒计时遮罩层
