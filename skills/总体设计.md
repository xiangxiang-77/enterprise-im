## 四端职责

### 后端 (im-server/)
- HTTP REST API (`/api/*`)
- TCP Socket 服务 (端口 9000，JSON 帧协议)
- WebSocket 推送 (`/ws`)
- 用户认证 (JWT)、好友、群组、消息、文件、通话控制
- Admin API (四角色权限矩阵: SUPER_ADMIN/OPERATOR_ADMIN/SECURITY_AUDITOR/READONLY_OPS)
- 外部 Provider 适配: SMS/SSO/Push/OCR/Office Preview

### Web 管理端 (im-ui/)
- 后台管理面板: 概览、用户管理、组织架构、组管理、资源管理、高级配置
- 用户端: 登录/注册、会话列表、聊天、联系人、文件管理、全局搜索、设置
- 组件库: shadcn/ui (Radix), recharts (图表), Tailwind CSS

### Qt 桌面端 (qt-client/)
- 单例主窗口，左侧面板(会话/联系人/搜索/文件/组织架构) + 右侧聊天区
- TCP Socket 直连，JSON 帧协议 (AUTH/TEXT/TYPING/ACK/CALL_*)
- SIP 音视频通话 (PJSIP/PJSUA 外部进程)
- QSettings 持久化 + SQLite 本地缓存
- VS2017 + Qt 5.9.3 + WebEngine (MingW备选)

### Flutter 移动端 (flutter-client/)
- Android 原生集成 PJSIP2 (JNI)
- TCP Socket 通信 (与后端端口 9000)
- HTTP REST API 调用
- 底部导航: 消息/通讯录/工作台/我
- SharedPreferences + sqflite 本地存储

## 通信协议
- HTTP REST: JSON, JWT Bearer Token 认证
- TCP Socket: 换行分隔 JSON 帧 (`{...}\n`), 12 种帧类型
- WebSocket: STOMP over SockJS
- SIP: PJSIP 信令 + RTP 媒体流

## 基础设施
- Docker Compose 一键部署 (9 服务): im-server, mysql, redis, minio, asterisk, coturn, pjsip-gateway
- Flyway 数据库迁移 (V1~V20)
- 构建: Maven, Vite, qmake+nmake, flutter build
