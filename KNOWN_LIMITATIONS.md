# 已知限制清单

> 本文档与功能清单、后台管理清单对照，标注当前版本已知未实现或需外部配置才能启用的功能。

## 一、需要外部 API Key 才能启用的功能

代码接口和适配器已完整实现，填入第三方服务密钥即可启用。未配置时走 demo/mock 通道。

| 功能 | 配置位置 | 需提供的密钥 |
|------|----------|-------------|
| 短信验证码(阿里云) | `application.yml` → `im.sms.aliyun.*` | AccessKey ID/Secret + 签名 + 模板 |
| 短信验证码(腾讯云) | `application.yml` → `im.sms.tencent.*` | SecretId/Key + SDK AppId + 签名 + 模板 |
| 推送(FCM) | `application.yml` → `im.push.fcm.*` | Firebase 服务账号 JSON |
| 推送(APNs) | `application.yml` → `im.push.apns.*` | p8 密钥 + Key ID + Team ID |
| OCR 文字识别(百度) | `application.yml` → `im.ocr.baidu.*` | API Key + Secret Key |
| Office 文件预览(OnlyOffice) | `application.yml` → `im.office.onlyoffice.*` | Document Server URL + Secret |
| SSO 单点登录(OIDC) | `application.yml` → `im.oidc.*` | issuerUri + clientId + clientSecret |

## 二、Web 前端长尾 UI 简化项

以下功能有基础实现但未达到功能清单要求的完整交互体验：

### 会话/聊天
- **网络状态提示**：状态字段存在但无顶部状态条 UI
- **链接预览卡片**：URL 可点击跳转，但无标题/描述/缩略图卡片展开
- **图片网格布局(1-9张)**：多张图片分别渲染，未合并网格
- **名片消息发送**：接收渲染正常，发送时使用硬编码 demo 数据，无联系人选择器
- **群公告置顶卡片**：公告仅在设置页查看，聊天页顶部无折叠公告卡片

### 图片查看器/编辑器
- **双指缩放(Pinch-to-Zoom)**：仅按钮缩放，无手势
- **左右滑动切换**：仅箭头按钮，无 swipe 手势
- **裁剪工具**：裁剪 Tab 存在但无裁剪框/区域选择交互
- **涂鸦画笔**：未实现
- **发送进度**：选择图片后直接发送，无上传进度条
- **原图选项**：未实现

### 文件
- **下载进度**：使用模拟进度(setInterval)，非真实传输进度
- **断点续传**：未实现 Range 分块下载
- **失败自动重试**：失败后需手动重发

### 群聊设置
- **群头像编辑**：无上传/编辑入口
- **群号/二维码**：占位图标，无生成逻辑
- **@所有人**：群公告编辑器无此能力
- **查找聊天记录按分类筛选**：跳转到搜索页但无分类过滤
- **投诉**：alert 占位，无表单提交
- **聊天背景**：6 个硬编码色块 + alert，无持久化

### 通讯录
- **官方账号入口**：User 类型有 isOfficial 字段但无列表入口
- **最近访问**：未实现
- **我的部门**：未实现

### 好友申请
- **扫一扫添加**：主菜单有扫一扫（入群），好友申请页无扫一扫添加入口

### 个人名片
- **设置备注编辑**：仅显示文本，无编辑 Dialog
- **解除好友/黑名单**：仅在单聊设置页操作，名片页无入口

### 全局搜索
- **匹配文字高亮**：搜索结果无高亮
- **历史搜索记录**：无存储/展示/清除
- **语音搜索**：未实现

### 通知设置
- **免打扰时间段选择器**：显示静态文本，无交互式 TimePicker

### 通用设置
- **独立关于页面**：版本号直接嵌在设置页，无独立关于页

### 登录
- **企业 SSO 入口**：登录页无 SSO 按钮，后端 OIDC 接口已就绪

## 四、Qt/Flutter 原生客户端说明

Qt 桌面端和 Flutter Android 端覆盖核心 IM 链路(登录→会话→消息→文件→通话)，但长尾 UI 不如 Web 丰富：
- 群设置/单聊设置提供开关型配置(mute/pin/DND等)，不含头像编辑、二维码等高级 UI
- 图片编辑器有旋转/灰度/裁剪/涂鸦，不含滤镜预设、文字叠加等高级编辑
- 搜索不含分类标签、历史记录、语音输入
- 不含引导页、扫码、官方账号等辅助功能

这是有意分工：后台管理仅 Web 端提供；原生端聚焦消息和通话核心体验。
