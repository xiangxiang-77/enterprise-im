# Skill: 移动端开发

## 目标

用 Flutter 实现 Android IM 客户端并交付 apk。

## 技术要求

- Flutter Android。
- TCP Socket 接服务端协议。
- 本地缓存 SQLite/Hive。
- 平台能力：通知、相册、相机、录音、文件选择、截屏监听。

## MVP 页面

- 启动/登录。
- 会话列表。
- 单聊/群聊。
- 通讯录。
- 好友申请。
- 图片/文件。
- 设置。

## 开发顺序

1. Flutter 工程骨架。
2. 协议模型和 TCP 客户端。
3. 登录、会话、消息。
4. 图片/文件/语音。
5. 通知和截屏事件。
6. apk 打包。

## 完成标准

- apk 安装成功。
- Android 真机或模拟器可登录、收发消息。
- 后台/锁屏重连策略可验证。
