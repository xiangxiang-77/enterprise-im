# Skill: 后台管理界面开发

## 目标

实现企业 IM 后台管理。

## 页面模块

- 数据大盘。
- 企业组织架构管理。
- 员工账号管理。
- 好友关系管理。
- 群聊高级管理。
- 消息内容管控与审计。
- 文件/图片/素材资源管理。
- 工作台应用配置。
- 系统基础配置与版本更新。
- 角色权限和操作日志。

## UI 规则

- 管理端要密度高、扫描快、少装饰。
- 列表必须有搜索、筛选、分页、导出按需。
- 高危操作用确认弹窗。
- 安全审计只读和可修改权限必须分离。

## 完成标准

- 管理员能完成组织、用户、群、消息、资源、配置的核心操作。
- 操作写入审计日志。
- 权限不足时按钮隐藏或禁用，API 也拒绝。
