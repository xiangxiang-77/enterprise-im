# Skill: 桌面端开发

## 目标

用 Qt 5.9.3 实现 Windows 桌面 IM 客户端并交付 exe。

## 技术要求

- Qt 5.9.3。
- TCP Socket 接服务端协议。
- 本地缓存 SQLite。
- 文件选择、下载、通知、托盘图标。

## MVP 页面

- 登录。
- 会话列表。
- 单聊/群聊。
- 联系人。
- 文件消息。
- 设置。

## 开发顺序

1. Qt 工程骨架。
2. TCP 客户端和协议编解码。
3. 登录和会话列表。
4. 消息收发、ACK、重连。
5. 文件/图片。
6. 打包 exe。

## 完成标准

- exe 可安装或直接运行。
- 两个桌面客户端互发消息成功。
- 断线重连后消息不丢。
