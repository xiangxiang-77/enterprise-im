# Skill: 深化设计

## 目标

把总体设计细化到接口、状态机、异常处理、权限。

## 必做设计

- 登录状态机：未登录、登录中、已登录、Token 过期、被踢下线。
- TCP 连接状态机：未连接、连接中、已连接、鉴权中、重连中、离线。
- 消息状态机：本地创建、发送中、服务端已收、对端已达、已读、失败、撤回、编辑。
- 群成员权限：群主、管理员、普通成员。
- 后台角色权限：超级管理员、运营、安全审计、运维只读。
- 文件状态：选择、上传中、已上传、发送中、可下载、下载中、失败、过期。
- 音视频状态：呼叫中、响铃、通话中、挂断、拒绝、超时、忙线、失败。

## 输出模板

- 业务规则。
- 状态机。
- 接口/事件。
- 数据读写。
- 错误码。
- 验收用例。

## 完成标准

- 开发者不用猜边界条件。
- 每个失败场景有 UI 和服务端处理。
