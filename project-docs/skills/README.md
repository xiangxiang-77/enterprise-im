# 项目 skills 库

用途：给 Codex/开发者按固定流程推进企业 IM 项目，避免只做 UI mock。

Skill 列表：

- `01-requirements-breakdown.md`：需求拆解。
- `02-overall-design.md`：总体设计。
- `03-deep-design.md`：深化设计。
- `04-database-design.md`：数据库设计。
- `05-key-technical-path.md`：关键技术路径设计。
- `06-server-development.md`：Java 服务端开发。
- `07-admin-web-development.md`：后台管理界面开发。
- `08-desktop-qt-development.md`：Qt 5.9.3 桌面端开发。
- `09-mobile-flutter-development.md`：Flutter 移动端开发。
- `10-automated-testing.md`：自动化测试。
- `11-multi-agent-collaboration.md`：多智能体协作。

所有 skill 都必须遵守：

- 先读 `codex.md`。
- 先确认当前真实进度。
- 新增功能必须带验证方式。
- 不把 mock UI 当完成。
- 涉及协议/DB/API 必须同步文档。
