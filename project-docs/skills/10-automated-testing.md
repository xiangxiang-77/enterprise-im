# Skill: 自动化测试

## 目标

用测试证明系统可交付。

## 测试层级

- Java 单元测试：service、mapper、权限规则。
- Java 集成测试：API、DB、Redis、对象存储。
- TCP 协议测试：鉴权、心跳、ACK、重连、离线消息。
- Web 测试：构建、关键页面冒烟、登录/聊天流程。
- 后台测试：权限、审计、高危操作。
- Qt/Flutter：核心流程手测 + 可自动化部分。
- 音视频：信令状态、PJSIP 呼叫、coturn 可达。

## 必测用例

- 单聊文本端到端。
- 群聊消息 fanout。
- 撤回 3 分钟规则。
- 群主/管理员撤回成员消息。
- 已读未读回执。
- 文件上传失败重试。
- Token 过期和被踢下线。
- 断网重连和离线消息补拉。

## 完成标准

- 一条命令可跑服务端测试。
- Web 构建通过。
- 有交付前冒烟清单和结果。
